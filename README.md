# Dice Game

A simple dice game built with Next.js, TypeScript, and Material UI where players predict whether a dice roll will be over or under a chosen threshold.

## Features

- Set a threshold value (1-100) using an interactive slider
- Choose "Over" or "Under" prediction
- Roll the dice and get results from 1 to 100
- Visual feedback for win/loss outcomes
- Game history tracking (last 10 rounds)
- Clean and responsive UI with Material UI components

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type-safe development
- **Material UI** - UI component library
- **Emotion** - CSS-in-JS styling

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
